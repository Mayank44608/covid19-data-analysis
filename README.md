# COVID-19 Data Analysis using Python

This project analyzes COVID-19 datasets to identify trends in confirmed cases, deaths, and recoveries.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
1. Data loading and cleaning
2. Exploratory Data Analysis
3. Visualization of COVID trends

## Insights
- Top affected countries
- Global case growth
- Time-series analysis
